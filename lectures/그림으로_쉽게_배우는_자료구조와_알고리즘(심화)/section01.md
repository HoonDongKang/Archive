## 💻 개요

### 📌 P-NP

문제가 어려운지, 쉬운지 혹은 해결 가능한지 불가한지 판단

#### 결정 문제

문제에 대해 Yes, No로 대답이 가능해야 함

#### 최적화 문제

최적의 해를 찾을 수 있는 문제

대부분의 최적화 문제는 결정 문제가 된다.

#### P

결정문제에 대해 결정론적 튜링 머신을 통해 다항 시간 내에 답을 구할 수 있는 문제

-   결정론적 튜링 머신: 다음 상태가 유일한 상태로 결정
    -   현재 > 다음 상태로 이동할 때 `if/else`여도 유일한 상태임
-   다항 시간: 문제 해결 시간을 다항식으로 표현이 가능 O(n^2)

상수 시간, 로그 시간으로 풀 수 있다 = P문제

#### NP

결정문제에 대해 비결정론적 튜링 머신을 사용하여 다항 시간 내에 답을 구할 수 있는 문제

-   비결정론적 튜링 머신: 현실에 존재 X 무한한 컴퓨팅 능력이 있다고 가정

#### NP-Hard

NP문제들이 있을 때 이 모든 NP 문제들을 다항 시간 내에 어떤 문제를 A로 환원시킬 수 있는 문제

#### NP-Complete

NP-Hard이면서 NP에 포함되는 문제
해결이 가능하다고 알려진 문제 <-> NP-Hard는 해결 가능한 지 불확실
