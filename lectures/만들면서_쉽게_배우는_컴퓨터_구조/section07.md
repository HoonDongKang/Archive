## 💻 메모리

### 📌 조합 논리회로

입력에 대해서만 출력하기 때문에 상태를 기억할 수 없음

-   기본 게이트, 멀티플렉서, 디코더, 반가산기, 전가산기, ALU,,

-   순차 논리회로(메모리 역할)
    출력을 다시 조합 논리회로의 입력으로 들어가야지 현재 상태를 만들어서 다음 상태를 결정시킬 수 있음

1. SR Latch
   | S | R | Q (다음 상태) | 설명 |
   | - | - | --------- | ----------- |
   | 0 | 0 | Q (유지) | 이전 상태 유지 |
   | 0 | 1 | 0 | Reset (Q=0) |
   | 1 | 0 | 1 | Set (Q=1) |
   | 1 | 1 | 무효 (X) | 금지 상태 (불안정) |

2. D latch
   | EN (CLK) | D | Q (다음 상태) | 설명 |
   | -------- | - | --------- | ----------- |
   | 0 | X | Q (유지) | 출력 유지 |
   | 1 | 0 | 0 | D의 값을 Q에 저장 |
   | 1 | 1 | 1 | D의 값을 Q에 저장 |

3. JK latch
   | J | K | CLK | Q (다음 상태) | 설명 |
   | - | - | --- | --------- | ----------------- |
   | 0 | 0 | 1 | Q (유지) | 상태 유지 |
   | 0 | 1 | 1 | 0 | Reset |
   | 1 | 0 | 1 | 1 | Set |
   | 1 | 1 | 1 | Q' (반전) | 현재 상태 반전 (Toggle) |
   | X | X | 0 | Q (유지) | CLK=0이면 상태 유지 |˚v
